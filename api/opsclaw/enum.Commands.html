Estop

Engage, inspect, and resume emergency-stop states.

Examples:

  • opsclaw estop
  • opsclaw estop --level network-kill
  • opsclaw estop --level domain-block --domain "*.chase.com"
  • opsclaw estop --level tool-freeze --tool shell --tool browser
  • opsclaw estop status
  • opsclaw estop resume --network
  • opsclaw estop resume --domain "*.chase.com"
  • opsclaw estop resume --tool shell

Fields

§estop_command: Option<EstopSubcommands>
§level: Option<EstopLevelArg>

Level used when engaging estop from opsclaw estop.

§domains: Vec<String>

Domain pattern(s) for domain-block (repeatable).

§tools: Vec<String>

Tool name(s) for tool-freeze (repeatable).
