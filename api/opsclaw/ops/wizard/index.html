Module wizard 

Source
Expand description

Composable interactive wizards for adding projects, environments, and targets. The three CLI verbs (config project add, config env add, config target add) all call into this module at different entry points; each wizard offers to walk into the next step inline so users don’t have to remember the right command order.

Hierarchy: project > environment > target. Targets always live under an environment; environments always live under a project.

prompt_new_target 🔒
Prompt for a fresh target’s connection details, autonomy, and optional context, then push it under project_name::env_name. Does not save.

run_project_add
config project add — create a project, then optionally chain into env + target creation. End state: as much of the project/env/target trio as the user opted in to, all written in one save.
