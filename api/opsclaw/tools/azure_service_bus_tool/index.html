Azure Service Bus tool.

Uses SAS-token auth against the namespace’s Service Bus REST surface (https://<namespace>.servicebus.windows.net). Read-write: list queues/topics/subscriptions, read message counts, peek (destructive — see comment), complete, dead-letter. Writes gated by autonomy.
